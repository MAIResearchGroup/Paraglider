======== Русский
В данном каталоге располагаются сторонние функции, выполнение которых необходимо в любом месте модели. Для выполнения достаточно скопировать файл функции в данную дерикторию и вызвать в нужном месте.

АХТУНГ!!! Названия файлов желательно давать говорящие, чтобы не приходилось по содержимому определять его назначение.
Если необходимо использование файлов из вложенных дерикторий в исполняемой функции, воспользуйтесь функцией MatLab addpath('путь к файлу').
